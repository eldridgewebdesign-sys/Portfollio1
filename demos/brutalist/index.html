01 / 04Manifesto

We don't decorate. We construct.

Most sites hide their structure behind soft gradients and stock polish. We show the grid, the type and the seams on purpose — because a site that's honest about how it's built is a site you can trust.

A —

Structure first

We start from the grid and the words, not a moodboard. The layout is the design — everything else is paint.

B —

Nothing hidden

Visible borders, real type, honest spacing. No fake depth, no glass, no glow. What you see is what ships.

C —

Fast & raw

Hand-built, lightweight, accessible. No template bloat, no library you'll never use. It loads and it works.
